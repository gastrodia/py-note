# 整数/浮点数运算

## 加
```python
print(1 + 1)

print(1.0 + 1.0)

print(1 + 1.0)
```
## 减
```python
print(1 - 1)

print(1.0 - 1.0)

print(1 - 1.0)
```

## 乘
```python
print(2 * 2)

print(2.0 * 2.0)

print(2 * 2.0)
```

## 除
**无论操作数类型，除法结果始终为浮点数**
```python
print(9 / 3)

print(9.0 / 3.0)

print(9 / 3.0)
```

## 整数除法运算符（向下取整除法）
```python
print(10 // 3)

print(10.1 // 3)
```
## 余数运算
```python
print(10 % 3)

print(10.0 % 3)
```

## 幂运算
```python
print(3 ** 2) # 3的平方

print(3.0 ** 2)
```

## 整数转浮点数
```python
print(float(1))
```

## 浮点数转整数
```python
print(int(1.2))
print(int(1.9))
```
